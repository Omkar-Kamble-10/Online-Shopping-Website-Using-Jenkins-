How to Customize

  1. Open index.html and about.html.
  2. Search for "Online Shopping "
  3. Change headings and paragraphs to match your content.

Example Uses

  • Event: Use cards for schedule, speakers, and registration.

Tech Stack

This project is built using simple technologies so it is easy to deploy with Jenkins:
